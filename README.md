# Water level sensor with 2 relais 230v, within webserver controls

## Hardware

* Arduinno Nano
* Ethernet Shield
* Double relais 5v
* RJ45 cable
* Water level switches
* A lot of patience...

## Wiring

Level sensors : 
* 0 => Blue
* 20 => Orange
* 40 => Orange - White
* 60 => Green - White
* 100 => Blue - White


* GND => Green
